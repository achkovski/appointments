# Appointments Management System

A modern appointment management system built with React and Node.js, designed for general business use to simplify the process of booking and managing appointments between users and service providers.

## Project Structure

- **backend/** - Node.js/Express API server
- **frontend/** - React application
- **docs/** - Project documentation

## Tech Stack

- **Frontend**: React, Vite
- **Backend**: Node.js, Express
- **Database**: PostgreSQL (Neon)
- **Authentication**: JWT

## Getting Started

See [project-specs.md](./project-specs.md) for detailed project specifications.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- PostgreSQL database (Neon account)
